## Naruto Shuriken 3D Model

This repository contains a 3D model of the regular (non-Fuma) Shuriken from the Naruto universe, designed in SolidWorks and exported as a 3MF file, and an STL file, as well as an SLDPRT file. This model can be used for 3D printing.

- Features:

**High-detail 3D model of the classic Shuriken,**

**Designed in SolidWorks for precision and scale,**

**Ready for 3D printing (3MF and STL formats).**

- Installation/Usage:

**1. Download either the 3MF or STL files onto your computer.**

**2. Open the NarutoShuriken.3MF/.STL file in your preferred slicer of choice.**

**3. Adjust the print settings based on your printer's specifications.**

**4. Start printing!**
